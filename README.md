# Wordle

A browser-based word guessing game. Guess the 5-letter word in 6 tries.

## How to Play

1. Type any 5-letter word and press Enter
2. Green = correct letter, correct position
3. Yellow = correct letter, wrong position  
4. Gray = letter not in the word

## Features

- Fullscreen responsive design (works on mobile, tablet, desktop)
- Visual keyboard with color feedback
- Flip animations for tile reveals
- Daily word that changes every day
- Play again with random words

## Run Locally

Just open `index.html` in your browser. No build step needed.
